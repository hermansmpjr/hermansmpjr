## Current Focus Areas

### Governed Multi-Agent Architecture (Copilot Studio)
I design and govern multi-agent systems in Microsoft Copilot Studio for industrial workflows. This includes governance layers, agent factories, knowledge discovery/mapping, and GitHub-centered documentation with controlled publishing.

The objective is to move from one-off agent experiments to governed, documented, and reusable agent systems.

### Document Intelligence & Parsing Pipelines
I build focused Python-based pipelines that convert customer-provided RFQs and technical specifications into structured, evidence-backed outputs, including concise briefs for engineers, canonical JSON artifacts, and operational run summaries.

Emphasis is placed on evidence preservation, conflict detection, missing-input reporting, and clean handoff to downstream agents or human reviewers.

### Application Engineering Agents
I develop families of Copilot Studio agents supporting RFQ intake, source diagnostics, engineering packet extraction, compliance matrix support, quote-pack reconciliation, and proposal assistance.

Key design principles include source discipline, routing control, missing-input handling, and structured output contracts.

### RFQ & Quote Workflow Automation
I build and maintain automations that connect quoting work across Salesforce, SharePoint, Teams, Outlook, Power Automate flows, Power BI dashboards, and Python-based document processing.

### Industrial Automation Support
Support for crane and hoist control systems, Magnetek drives and automation products, VFD motion control, HMI concepts, safety interlocks, and engineered material-handling applications.
